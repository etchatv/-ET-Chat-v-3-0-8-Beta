Willkommen bei der Installation von ET-Chat v3

Bevor die Datenbank angelegt werden kann, müssen Sie die Datei config.php mit einem Editor (Notepad oder Wordpad) auf ihrem PC öffnen und dort die Parameter Ihrer Datenbankverbindung eintragen. Diese Parameter erfahren Sie bei Ihrem Webhostanbieter.

Die aktuellen Einstellungen in der config.php :

Die Anbindung an die Datenbank erfolgt über _usedDatabaseExtension); ?>-Extension.

Als Datenbank ist _usedDatabase; ?> augewählt.
Host: _sqlhost; ?>
Login: _sqluser; ?>
Pass: _sqlpass; ?>
Database: _database; ?>
(Hier müssen die Parameter Ihres Datenbanks stehen.)
